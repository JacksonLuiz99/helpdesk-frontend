Help Desk Um sistema de chamados

Olá! Seja bem-vindo ao sistema Help Desk, desenvolvido por mim durante o curso de Desenvolvedor Full Stack.

Este sistema foi projetado para gerenciar chamados e simula um Help Desk básico, utilizando as tecnologias Angular 12 para o front-end e Spring Boot 2 para o back-end. O projeto está hospedado na nuvem, com o front-end rodando na Vercel e o back-end no Heroku.

O principal objetivo do projeto é demonstrar conceitos de aplicações SPAs (Single Page Applications), além de abordar práticas de autenticação e autorização utilizando Tokens JWT.

Você pode acessar o código fonte do projeto a qualquer momento através do link do GitHub disponível no menu lateral.

Fique à vontade para explorar o sistema e compartilhar suas sugestões de melhoria!
